# Data

This project focuses on the April-May 2020 SGR 1935+2154 campaign.

## `raw/`

Place downloaded HEASARC FITS or event files here.

The manuscript reproducer currently expects the NICER event files:

- `ni3020560101_cl.evt.gz`
- `ni3020560102_cl.evt.gz`
- `ni3020560103_cl.evt.gz`
- `ni3020560104_cl.evt.gz`

## `manifests/`

Observation manifests generated by `pipeline.fetch.save_campaign_manifests()`
are written here by default.
